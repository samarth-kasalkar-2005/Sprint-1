# 📊 AI Job Market Dashboard

An interactive **Power BI Dashboard** built using an AI Job Market dataset to analyze trends in salaries, job roles, skills, locations, experience levels, and employment types. The project demonstrates data cleaning, transformation, analysis, and visualization using **Microsoft Excel** and **Power BI**.

---

## 🚀 Project Overview

The AI Job Market Dashboard helps users explore and understand hiring trends in the AI industry by providing interactive visualizations and insights.

The project includes:
- 📁 Data preprocessing in Microsoft Excel
- 📊 Interactive dashboard in Power BI
- 📈 KPI cards and charts
- 🎛️ Slicers for dynamic filtering
- 📌 Business insights from the dataset

---
